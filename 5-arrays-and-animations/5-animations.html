You can create animations using spritesheets that have been loaded in.

Add this code in the createFunction():

player = game.add.sprite(200, 100, 'troll');
player.animations.add('left', [4,5,6,7], 10, true);

Adding the sprite works the same as loading a regular image. Adding the animation is a little trickier, since it has multiple arguments. Here's what they mean:

  1. A string that represents the name of the animation
  2. An array with the frames you want to use from the spritesheet. The frames go from left to right, like you're reading a book
  3. The speed of the animation. The higher this number, the faster the animation will play
  4. Whether or not the animation should lose

You can run an animation (usually in the updateFunction()) with:

player.animations.play('left');

The only argument is the name of the animation you want to play:

You can stop all the animations on a sprite with:

player.animations.stop();
