Arrays are lists of items in JavaScript. You can store a list of pretty much any combination of things, including strings, numbers or even other lists.

Here's what an array looks like in JavaScript:

  var fruits = ['apples', 'oranges', 'bananas'];

You can pull out an item from the array by adding [] with a number:

  console.log(fruits[1]);

Link to code

Since JavaScript starts counting at 0, the code above prints 'oranges' instead of 'apples'.
