As long as the Phaser JavaScript is loaded in, this skeleton will help you get started creating games using Phaser.

The first line generates the game screen, which is 400 pixels high and 400 pixels wide. The preloadFunction() loads in files you might need for your game, like images. The createFunction() sets up the game by letting you do things like place characters on the screen or add gravity. The updateFunction() runs 60 times per second and lets you do things like move characters around on the screen.
