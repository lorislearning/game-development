You add enemy sprites the same way you add player sprites. If you wanted a player and an enemy to interact, you could add this code to your updateFunction():

game.physics.arcade.overlap(player, baddie, hitBaddie, null, this);

The hitBaddie() function is run whenever a player sprite overlaps a baddie sprite.

Here is what the hitBaddie() function looks like. It's defined as a completely separate function, outside any of the pre-built Phaser functions:

function hitBaddie(player, baddie) {
  score = 0;
  text.text = 'Score: 0';
  player.kill();
}
