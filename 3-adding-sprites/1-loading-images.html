Add the following line's to your preloadFunction() and createFunction():


  function preloadFunction() {
    ...
    game.load.image('troll', 'http://i.imgur.com/Vm4PZ4h.gif');
  }

  function createFunction() {
    game.add.sprite(200,100, 'troll');
  }

Link to code

The image() method takes in two arguments: a name and a link to an image. It loads the image into your game. Both arguments are strings.

The sprite() method takes in three arguments: an x-value, a y-value and a name. The x-value and y-value places the sprite on a specific place on your game screen and the name should match up with an image from preloadFunction(). The first two arguments are numbers and the last argument is a string.
