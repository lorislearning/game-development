You can scale a sprite with:

  someSprite.scale.setTo(xMultiplier,  yMultiplier);

Here it is in context:

  function createFunction() {
    var player = game.add.sprite(200,100, 'troll');
    player.scale.setTo(2,2);
  }

Link to code

You first need to access the the scale property then call setTo() on the player's scale. The setTo() method takes in two arguments: the first is a multiplier for the width of the sprite and the second is a multiplier for the height of the sprite. Both arguments need to be numbers.
