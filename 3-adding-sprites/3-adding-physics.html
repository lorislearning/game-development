Add the following lines:


  // Addd this line above the createFunction()
  var player;

  function createFunction() {
    game.physics.startSystem(Phaser.Physics.ARCADE);
    player = game.add.sprite(200, 100, 'troll');
    game.physics.arcade.enable(player);
    player.body.gravity.y = -100;
    player.body.collideWorldBounds = true;

    ...
  }

You need to create a player variable outside the createFunction() so you can access it in the other functions too. The first in the createFunction() line adds the "arcade" physics system to your game. The third line adds physics to the player object. The fourth line adds gravity in the "y" direction. The last line makes sure the player object can't go past the edge of the game.
