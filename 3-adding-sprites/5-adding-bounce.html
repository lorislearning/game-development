You can get sprites to bounce with:

  someSprite.body.bounce.setTo(someMultiplier);

Here it is in context:

  function createFunction() {
    ...
    player.body.collideWorldBounds = true;
    player.body.bounce.setTo(0.8);
    ...
  }

Link to code

If you set the bounce multiplier to less than 1, the sprite will bounce shorter and shorter distances.
