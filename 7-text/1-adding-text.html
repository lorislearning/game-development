You can add text with the following code:

game.add.text(10, 10, 'Lives : ', { font: '34px Arial', fill: '#fff' });

The text() method takes in four arguments:

  1. The distance of the text from the top of the Phaser display
  2. The distance of the text from the left of the Phaser display
  3. The actual text that gets displayed
  4. A JavaScript object that customizes how the text looks
