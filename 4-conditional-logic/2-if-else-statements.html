if...else statements go one step further than if statements. They let you choose between two options, depending on if a condition is true or false. The following code pops up one alert only if the number variable is greater than zero and another alert if it's less than zero.

  var number = -1;

  if (number > 0) {
    alert('greater than 0');
  } else {
    alert('not greater than 0');
  }
