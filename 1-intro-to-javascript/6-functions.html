Functions are just chunks of code that are grouped together. They're kind of like variables, except they only hold instructions, not data. They're also similar to methods, like window.open(), except they don't need to be tied to an object with a .

You can actually call alert as a regular function, instead of a method attached to the window object:

  alert("I'm a function!");

Alert is a function that's built into JavaScript, but one of the great things about programming is that you can extend the language however you want! Here's how you write your own function:

  function theName() {
    // your function's code
  }

Every function you create needs the function keyword, a name, parentheses and code that goes inside of { }. Here's an example of a function that says hello:

  function sayHello() {
    alert("What's up!");
  }

You can run this function with:

  sayHello();

It would be great if you could customize the function to say hello to a specific person. You can do this by adding an argument to the function. Arguments look like variables that live inside a function's parentheses. They're placeholders that get filled whenever a function is called. If you added an argument to the sayHello() function, it would look like:

  function sayHello(name) {
    alert("What's up, " + name + "!");
  }

name just stands for the actual name you use when you call the sayHello() function.

This code:

sayHello('Neal');

pops up an alert that says "What's up, Neal!"

You can add multiple arguments by separating them with commas. This function calculates the area of a rectangle:

  function area(height, width) {
    alert(height * width);
  }
  
  area(5,10);
