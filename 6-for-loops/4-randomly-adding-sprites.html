You can also use a loop with the Math.random() function to randomly add sprites on the screen. The Math.random() function generates a number between 0 and 1. If you multipied that result by 100, you woudl create a random number between 0 and 100:

Math.random() * 100;

In a game, you might randomize the locations of sprites:

grass = game.add.group();
grass.enableBody = true;
for(i=0; i < 16; i++) {
  // Randomly add sprites to on the screen
  var grassTile = grass.create(i*48, Math.random()*400, 'environment');
  grassTile.frame = 1;
  grassTile.body.immovable = true;
}
