If you ever find yourself repeating code, think about using a loop. For example, lets say you wanted to print out all of the items in an array. You could do something like this:

fruits = ['apples', 'oranges', 'bananas'];

console.log(fruits[0]);
console.log(fruits[1]);
console.log(fruits[2]);

But this code is pretty bad. You're repeating yourself a lot. One of the core principles in programming is Don't Repeat Yourself (DRY). You can use loops to DRY up our code

fruits = ['apples', 'oranges', 'bananas'];

for(i=0; i<3; i++) {
  console.log(fruits[i]);
}

Link to code

There's a lot going on here, so let's go through it step-by-step:

  • The for() function takes in three inputs separated by semi-colons
  • The first input is a variable with starting value. In this case, the variable is i and its starting value is 0
  • The second input is a rule for when the loop should keep running. In this case, the rule is when i < 3
  • The last input is what happens to the variable each time the loop runs. In this case, the variable i is increased by one each time.
  • The code between the { } gets run every time the loop runs
