Groups let you combine sprites together so that you can DRY up your Phaser code.Let 's say you wanted to group a set of sprites together. First you would need to create a variable for the group:

 var fruits;

Then you would need to create the group in the createFunction():

 fruits = game.add.group();

Then, assuming you wanted to add physics, you can add it to the entire group at once:

 fruits.enableBody = true;

Then you can create your sprites and add them to your group inside a loop. Suppose you wanted to create 3 sprites:

for(i=0; i<7; i++) {
  var item = fruits.create(i*30, 100, 'randomSprites ');
}

Finally, in the updateFunction(), you can collide a sprite with an entire group:

 game.physics.arcade.collide(player, fruits);
